Mind Move

Aplicativo voltado ao bem-estar de pessoas neurodivergentes, promovendo foco e equilíbrio por meio de rotinas físicas leves e personalizáveis, incentivando hábitos saudáveis de forma prática.

Principais tecnologias utilizadas: Java, Spring Boot e REST API.

Clima Tempo

Ferramenta interativa que fornece previsões meteorológicas detalhadas, permitindo ao usuário consultar a cidade desejada e visualizar dados estruturados da semana de forma clara e responsiva.

Principais tecnologias utilizadas: HTML, CSS, JavaScript e API Open-Meteo.

Garden Center

Plataforma dedicada a apoiar pequenos produtores rurais na venda de plantas, conectando cultivadores a clientes de forma simples e acessível. Facilita a divulgação e amplia oportunidades.

Principais tecnologias utilizadas:Java, Spring Boot e REST API.

Tá na Mesa

Plataforma de delivery, desenvolvida para facilitar o gerenciamento de produtos e oferecer uma experiência ágil aos usuários, com endpoints REST completos para operações de CRUD.

Principais tecnologias utilizadas: Java, Spring Boot e REST API.

Blog Profissional

Sistema completo para criação e gestão de conteúdos, permitindo administrar usuários, temas e postagens de forma organizada, proporcionando experiência interativa para autores e leitores.

Principais tecnologias utilizadas:React, Tailwind, CSS, Java e Spring Boot.

Smart RH

API robusta que simula um sistema de Recursos Humanos, oferecendo CRUD para Colaborador, Departamento e Usuário, além de um endpoint que calcula salários líquidos com precisão.

Principais tecnologias utilizadas: Java, Spring Boot e REST API.
